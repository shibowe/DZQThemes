# The Themes of Discuz Q!  
感谢关注本项目!  
DZQTehems收集并整理Discuz新产品：Discuz Q!的主题模板.  

## 开发工具推荐(Develop Tools Recommended)  
推荐使用 [VSCode](https://code.visualstudio.com/) 进行开发，针对此项目VSCode插件推荐安装：
- Vetur
- ESLint
- Stylelint
- Prettie
- Vue VSCode Snippets
- JavaScript(ES6) Snippets ESDoc
